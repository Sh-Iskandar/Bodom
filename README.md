# BODOM — оптимизированный сайт

## Структура

```
bodom-optimized/
├── index.html   ← только каркас страницы (без повторяющихся блоков)
├── styles.css   ← стили (не трогать без нужды)
├── data.js      ← ВСЕ ДАННЫЕ САЙТА (правите только его)
├── main.js      ← рендер (не трогать без нужды)
└── images/      ← сюда положите фото
```

## Как добавить/изменить контент

Откройте **`data.js`** — там массивы:

- `NAV` — пункты меню
- `STATS` — 4 числа в hero-секции
- `SERVICES` — карточки услуг
- `EQUIPMENT` — таблица оборудования `[название, кол-во, описание]`
- `PROJECTS` — карточки проектов (с фото)
- `GALLERY` — галерея фото
- `CONTACTS` — блок контактов
- `FOOTER_COLS` — колонки в футере

### Пример: добавить 50 фото в галерею

Просто дописываете 50 строк в массив `GALLERY`:

```js
const GALLERY = [
  { src: "images/gallery-01.jpg", cat: "Литьё",   alt: "..." },
  { src: "images/gallery-02.jpg", cat: "Сварка",  alt: "..." },
  // ...
  { src: "images/gallery-50.jpg", cat: "Проекты", alt: "..." },
];
```

HTML менять не нужно — карточки сгенерируются автоматически.
Флаг `tall: true` делает карточку высокой (акцентной).

### Пример: добавить услугу

```js
const SERVICES = [
  // ...существующие...
  { icon: "★", title: "Новая услуга", text: "Описание услуги." },
];
```

## Как запустить

Просто откройте `index.html` в браузере (двойной клик).
Или через Live Server в VS Code.

> ⚠ Если фото не показываются при открытии двойным кликом —
> запустите через локальный сервер: `npx serve` или Live Server.
